# Portfolio

This is a Next.js portfolio project for Yugal Jakasaniya. It includes pages for projects, contact, resume (PDF available in `public/`), and other personal sections.

Quick links
- Resume PDF: `/Yugal_jakasaniya_resume.pdf` (in `public/`)

Getting started (local)
1. Install dependencies:
	```powershell
	cd D:\Project\Portfolio
	npm install
	```
2. Run development server:
	```powershell
	npm run dev
	```
3. Open `http://localhost:3000` and visit `/resume` to view the resume page.

Available scripts (from `package.json`)
- `npm run dev` — start dev server
- `npm run build` — build production bundle
- `npm run start` — start production server after build

Where to find things
- Pages: `app/` (Next.js App Router)
- Components: `components/`
- Public static files: `public/`

Deployment
- See `DEPLOY_VERCEL.md` for step-by-step instructions to deploy to Vercel (free tier).

Notes
